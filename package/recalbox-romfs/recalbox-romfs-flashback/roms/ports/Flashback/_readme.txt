## RECALBOX - SYSTEM FLASHBACK ##

Put your Flashback games in this directory.

Flashback games must have a ".map/.aba/.seq" extension.

Content that can be loaded by the REminiscence core have the following file extensions:

.map (DOS Map Data)
.aba (DOS (Demo) Map Data)
.seq (DOS CD Map Data)
//.lev (Amiga Map Data)

Fichiers de données:
- Fichiers de données de jeu PC
- Fichiers Amiga .mod (pour la musique)
- VOICE.VCE pour le dialogue dans le jeu (facultatif)
