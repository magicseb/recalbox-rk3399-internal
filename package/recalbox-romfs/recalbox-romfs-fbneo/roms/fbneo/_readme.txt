## RECALBOX - SYSTEM FBNEO ##

Put your FinalBurn Neo roms in this directory (cps3 support, updated version, but less performant than piFBA).

The roms version must be 0.2.97.44

Rom files must have an ".iso/.zip/.7z" extension.

Special files for fba libretro core :
- Add your samples files in /recalbox/share/bios/fbneo/samples/
